# 현대사회와 법 — 문제풀이 (GitHub Pages 배포용)

정적 사이트입니다. 빌드 과정이 없고 `index.html` 하나만 있으면 동작합니다.

- 총 681문항 (원본 481 + 자체 제작 O/X 200)
- 외부 의존성 없음 — CDN·폰트·이미지 요청이 전혀 없어 오프라인에서도 열립니다
- 학습 기록은 브라우저 localStorage에 저장됩니다 (기기·브라우저마다 분리)

## 배포 방법

1. 이 폴더를 GitHub 저장소로 올립니다
2. 저장소 Settings → Pages → Source를 `main` 브랜치 / `/ (root)` 로 지정
3. 몇 분 뒤 `https://<사용자명>.github.io/<저장소명>/` 에서 열립니다

## 기기 간 기록 옮기기

localStorage는 기기마다 분리되므로 노트북과 아이패드 기록이 자동으로 합쳐지지 않습니다.
시작 화면의 **기록 내보내기 / 불러오기**를 쓰세요.

1. 노트북에서 "기록 내보내기 (JSON)" → `법퀴즈_기록_YYYY-MM-DD.json` 다운로드
2. 파일을 아이패드로 옮김 (에어드롭·메일·클라우드 등)
3. 아이패드에서 "기록 불러오기" → 파일 선택 또는 내용 붙여넣기 → 병합/덮어쓰기 선택 → 실행

- **병합**: 문항별로 더 많이 푼 쪽(이력이 긴 쪽)을 채택. 같으면 불러온 파일을 따름
- **덮어쓰기**: 이 기기 기록을 버리고 파일로 완전히 교체
- 불러오기 직전 상태는 자동 백업되어 "직전 상태로 되돌리기"로 복구 가능

## 아이패드에서 앱처럼 쓰기

Safari로 열고 공유 → "홈 화면에 추가"를 누르면 주소창 없이 전체 화면으로 실행됩니다.
